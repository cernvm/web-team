CernVM-FS Package Repositories

yum

https://cvmrepo.web.cern.ch/cvmrepo/yum

In order to add the yum repository, run: 

sudo yum install https://ecsft.cern.ch/dist/cvmfs/cvmfs-release/cvmfs-release-latest.noarch.rpm

apt

https://cvmrepo.web.cern.ch/cvmrepo/apt

In order to add the apt repository, run: 

sudo apt-get install lsb-release
wget https://ecsft.cern.ch/dist/cvmfs/cvmfs-release/cvmfs-release-latest_all.deb
sudo dpkg -i cvmfs-release-latest_all.deb
rm -f cvmfs-release-latest_all.deb
sudo apt-get update
